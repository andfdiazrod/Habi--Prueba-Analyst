# Habi: Prueba Analyst
 Desarrollo de la prueba para la posición de analista de datos en Habi de Andrés Felipe Díaz Rodríguez

En el repositorio se encuentra un notebook de Jupyter llamado "Prueba Analyst.ipynb" en el que se hicieron todos los procesos de manejos de datos. gráficas, tablas y análisis estadístico para la solución del ejercicio en Python. También se encuentra el enunciado y la base de datos proporcionada para la solución del ejercicio y la solución en un pdf llamado "SOLUCIÓN - Andrés Felipe Díaz rodríguez - Habi - Prueba_Analyst.pdf"
